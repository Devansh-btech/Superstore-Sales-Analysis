# Superstore Sales Analysis Dashboard

## Project Overview
This project analyzes sales data from a retail superstore using Microsoft Excel.

The goal was to identify:
- Top-performing product categories
- Most profitable regions
- Sales and profit trends
- Impact of discounts on profitability

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Analysis
- Dashboard Creation

## Key Findings
- Technology generated the highest sales.
- Technology generated the highest profit.
- West region generated the highest profit.
- Furniture had the highest average discount.
- Furniture in the Central region showed losses.
